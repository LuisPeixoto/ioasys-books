## Instalação

### Clone do repositório

```bash
$ git clone https://github.com/LuisPeixoto/ioasys-book

$ cd ioasys-book
```

### Instalação das dependências

```bash
$ yarn install
```

### Execução

```bash
$ yarn android
```

ou

```bash
$ yarn ios
```
